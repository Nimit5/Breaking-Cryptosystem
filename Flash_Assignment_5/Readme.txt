Dependencies:

- numpy : support for calculations
- pexpect : used to connect to the server to generate ciphertext
- pyfinite : for Field operations

How to run:

Open the current directory in the terminal and type "make run" this command will first install all the packages/dependencies and then run the script inside it. Script firstly generates plaintext.txt and its corresponding ciphertext.txt. And using this files our python script will decrypt the password and display it the terminal.

Estimated script runtime : 5-6 minutes(atmost)
